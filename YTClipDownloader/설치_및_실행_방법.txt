========================================
  YTClipDownloader  설치 및 실행 방법
========================================

[ 실행 방법 ]

run.bat 실행
  - Python 및 필요한 패키지가 없으면 자동으로 설치 후 실행됩니다.
  - 인터넷 연결 필요 (최초 실행 시)
  - 이후에는 그냥 run.bat 만 실행하면 됩니다.


[ 자동 설치 조건 ]

- Python 자동 설치: Windows 10 버전 1709 이상 (winget 내장)
- 구형 Windows 또는 자동 설치 실패 시:
    1. https://www.python.org/downloads/ 에서 Python 수동 설치
    2. 설치 시 "Add Python to PATH" 반드시 체크
    3. setup.bat 실행 후 run.bat 실행


[ 폴더 구조 ]

YTClipDownloader\
├── main.py                  프로그램 본체 (코드)
├── setup.bat                패키지 수동 설치 스크립트
├── run.bat                  실행 스크립트 (Python 없어도 자동 설치)
├── requirements.txt         필요한 패키지 목록
├── 설치_및_실행_방법.txt    이 파일
├── download\                기본 저장 폴더 (자동 생성)
└── ytdlp\
    ├── ffmpeg.exe           영상 처리 도구
    ├── ffprobe.exe          영상 정보 분석 도구
    └── yt-dlp.exe           영상 다운로드 도구


[ 문제 해결 ]

Q. run.bat 실행 시 Python 자동 설치가 실패하는 경우
   - https://www.python.org/downloads/ 에서 직접 설치
   - 설치 시 "Add Python to PATH" 체크 필수
   - 설치 후 setup.bat 실행, 이후 run.bat 실행

Q. 영상 다운로드가 안 되는 경우
   - 도구 메뉴 → yt-dlp 업데이트 실행 (사이트 변경 대응)

Q. 미리보기가 안 되는 경우
   - setup.bat 을 실행하여 PyQtWebEngine 설치 확인
